# Canonical Project Model of MagnetarYggdrasil - Ruby (Rails)

## Purpose
This project exists to compare the Ruby on Rails framework against 34 other technologies by implementing the exact same application (MagnetarYggdrasil). It follows the **Magnetar Canonical Project Model** for strict governance, ensuring that the comparison is scientific and valid.

## How to Use This Repository

1.  **Clone the canonical model**: This repository structure is strict.
2.  **Configuration**: Check `projects/RubyInRails.project.yml`.
3.  **Documentation**: Refer to the files listed below.
4.  **Running**: See `README_RUN.md`.

All changes are logged in `BITACORA.md`.
