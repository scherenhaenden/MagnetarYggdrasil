# How to Run MagnetarYggdrasil - Ruby (Rails)

## Prerequisites
*   Ruby 3.2+
*   Bundler
*   SQLite3

## Installation
1.  Navigate to the directory:
    ```bash
    cd RubyInRails
    ```
2.  Install dependencies:
    ```bash
    bundle install
    ```

## Running the Application
```bash
rails server
```
The server will start on port 3000.

## Running Tests
```bash
bundle exec rspec
```
