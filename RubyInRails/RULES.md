# Canonical Ruleset of MagnetarYggdrasil - Ruby (Rails)

## Introduction
These rules codify the Magnetar standard.

## Naming Conventions
*   **Repositories:** `magnetar-ruby-rails` (internal ref).
*   **Branches:** `feature/`, `fix/`, `chore/`.
*   **Tasks:** `kebab-case`.

## Required Files
*   `README.md`, `PLAN.md`, `BITACORA.md`, `REQUIREMENTS.md`, `ARCHITECTURE.md`, `RULES.md`, `STATUS.md`, `TESTING.md`, `BLOCKERS.md`.
*   `projects/RubyInRails.project.yml`.

## Branching Conventions
*   `master`: Immutable release line.
*   `feature/*`: Feature branches.

## Allowed Task States
1.  `planned`
2.  `ready`
3.  `in_progress`
4.  `in_review`
5.  `blocked`
6.  `done`

## Blockers
Log in `BLOCKERS.md`.
